Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

I'm terrible at analogies, so I'm just going to tell you the Udacity one we were told during the lessons with them.

Html are the framework of the building, the css is the interior decoration. And then the Dom is like a tree where you can pluck the leaves off and rearrange them in a different pattern.

What is it meant by Boxifying a design?

Every website is made up of many boxes, some within others. Boxifying a design, means that one needs to design a website using boxes, some times even using boxes within boxes, to make up the website into a whole.

What is the box model?

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
