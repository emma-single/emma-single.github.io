Udacity Blog #2

What is grid based design?

Grid based design is the practice of dividing your webpage into a grid, using different arrangements of number of columns and widths.

Many websites use a grid with 12 columns as this gives quite a lot of flexibility. More than 12 columns however would make for too narrow columns.

What's all the hype about responsive webpages?

Responsive design means developing websites which will resize and adapt to different sizes of screen (different resolutions) and types of devices, ensuring the website is still legible and is looking great, no matter what device the viewer is on.

What is semantic structure?

Wikipedia defines it as: "...the use of HTML markup to reinforce the semantics, or meaning, of the information in webpages and web applications rather than merely to define its presentation or look."
